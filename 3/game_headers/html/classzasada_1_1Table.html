Detailed Description

template<class Key, class T, class Hash = std::hash<Key>, class KeyEqual = std::equal_to<Key>, class Allocator = std::allocator<std::pair<const Key, T>>>
class zasada::Table< Key, T, Hash, KeyEqual, Allocator >

A template class implementing a hash table with separate chaining for key-value pairs.

This class provides basic functionality for inserting, erasing, finding, and iterating through key-value pairs stored in a hash table.

Template Parameters
KeyType of the key used in the table.
TType of the value associated with the key.
HashHash function used to hash the keys.
KeyEqualFunction to compare the keys for equality.
AllocatorAllocator used for memory management.
