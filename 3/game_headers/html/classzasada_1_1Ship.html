Detailed Description

Represents a generic ship class with basic attributes and behaviors.

This class provides the basic attributes and functionality for ships including their name, captain, speed, health, position, and price. It includes methods for modifying and retrieving these attributes and basic ship behaviors like moving and taking damage.
