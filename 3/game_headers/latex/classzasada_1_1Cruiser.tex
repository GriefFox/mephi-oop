\doxysubsection{Detailed Description}
Represents a cruiser ship with weapon capabilities. 

This class is a derived class from \doxylink{classzasada_1_1Ship}{Ship} and \doxylink{classzasada_1_1AWeaponShip}{AWeapon\+Ship}, providing additional storage for weapons and the ability to attack.
