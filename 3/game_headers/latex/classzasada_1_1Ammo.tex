\doxysubsection{Detailed Description}
Represents ammunition used in weapons. 

The {\ttfamily \doxylink{classzasada_1_1Ammo}{Ammo}} class defines the properties of ammunition, including its name, price, storage size, current count, and total count in storage. It provides methods for managing the amount of ammo, including reducing, reloading, and calculating costs.
