\doxysubsection{Detailed Description}
A base class for all weapon types, encapsulating common functionality. 

Forward declaration of \doxylink{classzasada_1_1Weapon}{Weapon} class.

This class defines the common properties and methods for different types of weapons, including their damage, ammo, range, fire rate, reload time, and price.
